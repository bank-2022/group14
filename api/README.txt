## asennusohje
----------------
aja api kansiossa komento 'npm install'. Sitten sovellus käynnistyy komennolla 'npm start'. 
Tiedostojen muokkauksen jälkeen paina 'ctrl' + 'c' , jotta nodeprosessi tapetaan ja voit käynnistää sen taas 'npm start' komennolla.

## .env
----------------
Luo api kansioon tiedosto .env ja kirjoita sinne muuttuja nimeltään MY_TOKEN=''
Aja komentoriviltä komento 'node create_token.js' ja kopsaa "token" muuttujaan MY_TOKEN='x' kohtaan x.
